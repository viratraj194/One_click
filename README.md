**OneClick** is a high-performance safety application designed to bridge the gap between an emergency and the arrival of help. Built with a "single-tap" philosophy, it ensures that in critical moments, your location and distress signals reach the right people instantly.



---



## ✨ Key Features



- 🆘 **Instant SOS:** A single-tap interface to trigger emergency sequences.

- 📍 **Real-time GPS:** Broadcasts live location data to emergency contacts and authorities.

- ✉️ **Background SMS:** Automated messaging system that operates even under low bandwidth.

- 🛡️ **Verified Security:** Aadhaar-linked verification to ensure system integrity and prevent prank alerts.

- ⚡ **Cross-Platform:** High-performance UI built with Flutter for seamless use.



---



## 🛠️ Tech Stack



### Frontend

- **Framework:** Flutter

- **Language:** Dart

- **State Management:** Provider / Bloc

- **APIs:** Google Maps SDK, Geolocator



### Backend

- **Framework:** Django (Python)

- **Database:** PostgreSQL

- **Security:** JWT Authentication & Aadhaar Integration API
